Game Rules

  1. Clicking on cells cycles through the different element types, with options to rotate or delete using the right mouse button.
  2. After selecting a tile, elements are chosen from a palette, or you can switch between elements using the arrow keys.
  3. Elements can be dragged and dropped onto a tile from a palette.
  4. The railway line is drawn by holding down the mouse button and moving across the cells.
  5. Each cell has two exit points, and the train can enter each neighboring cell from the correct direction.
  6. Every cell that needs to be touched is touched.
  7. Every cell where no track can be placed remains empty.
